# 测试计划

**项目**：ChildPCGuard  
**文档编号**：TP-001  
**版本**：v1.0  
**日期**：2026-05-03  
**状态**：已基线

---

## 1. 测试目标

- 验证所有功能需求（SRS-001）按预期工作
- 验证自我保护机制对标准用户的有效性
- 验证系统在异常场景（崩溃、重启、断网）下的稳定性
- 验证每个 Phase 的交付物满足对应验收标准

---

## 2. 测试范围

### 2.1 纳入测试范围

- 时间计时与配额管理（TimeTracker、RuleEngine）
- 锁屏触发与解锁（LockOverlay）
- 自动关机机制（双重保障）
- 双进程互保（AgentA ↔ AgentB）
- 管理员界面功能（AdminPanel）
- 自我保护机制（DACL、ACL、注册表保护）
- 安装脚本（install.ps1）
- 崩溃恢复与状态持久化

### 2.2 排除测试范围

- Windows 系统内核兼容性测试（由 Microsoft 保证）
- .NET 8 运行时本身的正确性
- Phase 6 可选功能（待功能确定后补充测试用例）

---

## 3. 测试策略

### 3.1 测试分层

```
┌─────────────────────────────────┐
│         端到端测试（E2E）         │  手动执行，覆盖完整用户场景
├─────────────────────────────────┤
│         集成测试（Integration）   │  手动 + 脚本，覆盖进程间通信
├─────────────────────────────────┤
│         单元测试（Unit）          │  自动化，覆盖核心算法逻辑
└─────────────────────────────────┘
```

### 3.2 各层策略

| 层级 | 工具 | 执行方式 | 目标覆盖率 |
|------|------|----------|-----------|
| 单元测试 | xUnit + Moq | CI 自动运行 | TimeTracker、RuleEngine ≥ 80% |
| 集成测试 | 手动 + PowerShell 脚本 | 每个 Phase 结束时 | 全部 IPC 接口 |
| E2E 测试 | 手动（测试账户） | 每个 Phase 验收时 | 全部用户场景 |
| 防护渗透测试 | 手动（标准用户账户） | Phase 2 / Phase 4 结束 | 全部 S-xx 用例 |

### 5.4 防护渗透测试用例（PT）

> **执行账户**：标准用户（孩子账户）

| 用例 ID | 攻击手段 | 操作 | 期望结果 |
|---------|----------|------|----------|
| PT-01 | 任务管理器 Kill GuardService | 任务管理器 → 结束任务 | 操作被拒绝，或服务 10 秒内复活 |
| PT-02 | sc stop 停止服务 | `sc stop ChildPCGuard` | 返回"拒绝访问 (5)" |
| PT-03 | sc delete 删除服务 | `sc delete ChildPCGuard` | 返回"拒绝访问 (5)" |
| PT-04 | 删除安装目录文件 | 尝试删除 `%ProgramFiles%\ChildPCGuard` 中的文件 | 操作被拒绝 |
| PT-05 | 修改配置文件 | 尝试打开或修改 `config.bin` | 操作被拒绝 |
| PT-06 | 修改注册表服务键 | Regedit 尝试修改 `HKLM\SYSTEM\...\ChildPCGuard` | 操作被拒绝 |
| PT-07 | 修改系统时间 | 控制面板修改时间 | 标准用户无此权限；即使成功，NTP 检测后触发锁屏 |
| PT-08 | 结束 AgentA 进程 | 任务管理器结束 AgentA | 操作被拒绝，或 30 秒内 AgentB 重启 AgentA |
| PT-09 | 结束 AgentB 进程 | 任务管理器结束 AgentB | 操作被拒绝，或 30 秒内 AgentA 重启 AgentB |
| PT-10 | 锁屏期间 Alt+Tab 切换 | 锁屏界面显示时按 Alt+Tab | 无反应，锁屏保持 |
| PT-11 | 锁屏期间 Win+D | 锁屏界面显示时按 Win+D | 无反应，锁屏保持 |
| PT-12 | 禁用任务计划 | 任务计划程序禁用关机任务 | 操作被拒绝，或 GuardService 计时器仍可关机 |

### 5.5 异常与恢复测试用例（RT）

| 用例 ID | 场景 | 操作 | 期望结果 |
