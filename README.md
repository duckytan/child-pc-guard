# ChildPCGuard

> 一款运行在 Windows 10/11 上的儿童电脑使用时间管控程序，具备企业级自我保护能力。

---

## 项目简介

市面上的家长控制软件要么收费联网，要么对"会任务管理器、会 `sc stop`"的孩子形同虚设。ChildPCGuard 专为此场景设计：

- 每日使用时长限制（工作日 / 周末分别配置）
- 超时自动锁屏，需家长密码解锁
- 每天固定时间自动关机
- **12 层防护体系**：双进程互保 + 虚拟桌面锁屏 + 关机双保险 + 防调试 + 防卸载……

程序以系统服务形式运行，外观伪装为系统进程，不在任务栏显示。

---

## 功能亮点

| 功能 | 说明 |
|------|------|
| 时间配额 | 工作日 / 周末独立配置，到期自动锁屏 |
| 虚拟桌面锁屏 | 切换至独立虚拟桌面，任务栏 / 桌面全部隐藏，密码解锁 |
| 自动关机 | 每天指定时间强制关机（双保险：服务计时器 + 任务计划程序） |
| 双进程互保 | AgentA / AgentB 互相监控，任一被杀立即重启对方 |
| 防强杀 | SYSTEM 级权限保护，标准用户无法终止服务 |
| 防卸载 | 服务注册表写保护，安装目录 ACL 锁定 |
| 图形化管理 | AdminPanel 需密码登录，可修改规则 / 查看日志 / 临时延时 |
| 崩溃自恢复 | 进程意外退出后自动重启，重启后恢复上次计时状态 |

## 系统架构

```
GuardService (SYSTEM)          ← 核心时间管控服务
    ├── TimeTracker            ← 计时模块
    ├── RuleEngine             ← 规则引擎
    └── LockScreen             ← 虚拟桌面锁屏

AgentA / AgentB (SYSTEM)      ← 双进程互保守卫

AdminPanel (管理员)            ← 家长图形化管理界面

install.ps1                   ← 一键安装脚本（支持 -Silent 参数）
```

进程间通信：Named Pipe（SYSTEM 权限命名管道）

---

## 安装前提条件

1. **操作系统**：Windows 10 1903+ 或 Windows 11
2. **孩子账户**：必须是**标准用户**（非管理员）
3. **运行时**：.NET 8 Desktop Runtime（安装脚本自动检测并提示）
4. **安装者权限**：家长需以管理员身份运行安装脚本

---

## 快速安装

```powershell
# 以管理员身份打开 PowerShell
.\install.ps1
```

安装向导会引导您：
1. 设置管理员密码（建议 12 位以上，孩子不知道的密码）
2. 配置工作日 / 周末每日时长（分钟）
3. 设置每日自动关机时间

**静默安装（适合重复测试）：**

```powershell
.\install.ps1 -Silent -AdminPassword "YourPass" -WeekdayLimit 120 -WeekendLimit 240 -ShutdownTime "22:00"
```

## 技术栈

- **语言**：C# 12 / .NET 8
- **UI**：WPF（AdminPanel + 锁屏界面）
- **服务**：Windows Service（`sc.exe` 注册）
- **IPC**：Named Pipe
- **存储**：JSON 配置文件（AES-256 加密）+ SQLite 日志
